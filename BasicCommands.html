Basic Commands

Basic Commands

  • mkdir- creates a new directory; For example mkdir web, will create a directory call “web”
  • Ping- checks connectivity status to a server
  • ls - lets you know what files are inside the directory you are in.
  • Cd - takes you to a certain directory on your pc. Example cd Desktop
  • Cat - displays the content of your file. Example: cat index.html would result to “Hello World”
  • Hostname- allows you to know your name in your host or network, hostname -I gives your IP address
  • Df- shows the size, used space, and available space
  • Exit- this command allows to close a terminal windows, end the execution of a shell script, or log out of an SSH remote access session
  • History- lists the commands you have previously issued on the command line
  • mv- allows you to move files and directories form to directory
  • Passwd- lets you change the password for a user. Just type passwd to change your own password
  • Pwd - prints the working directory
  • Ssh- allows you to make a connection to a remote Linux computer and log into your account. To make a connection, you must provide your username and IP address.
  • How can you log into your docker container ? First step, you must get the IP address of Container with the docker inspect command with your container name. Next, you want to use the SSH command to log into the container. So “ssh root 192.168.1.1: for example.Finally input password and connection will be established.
