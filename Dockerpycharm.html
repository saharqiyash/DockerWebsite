Docker Pycharm

First,you must make sure you are using Pycharm pro and not community. You can do this by going to JetBrains and signing up for a student license. After you have done that, open your pycharm, go to VCS and get from version control a repo that has a python script or create a one. Next, you need to go into settings, go down to "Build,Execution,Deployment" and make sure your docker build is running by selecting Docker for Mac. After this, you are going to the top right of our program and click add configuration. In this configuration select the "+" button, go down to docker and then Dockerfile. Find out appropriate information and hit "OK". Once you the hit the play button it will your run you image.

Finally, make sure you add a remote interpreter. First, go to the way bottom right of your pycharm program, select the tab next to Git:master. Then select add interpreter, once selected go to docker and let it fill out the correct the information. Finally, hit "ok" and select this interpreter in the bottom right.
